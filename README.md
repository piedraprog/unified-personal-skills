## 🔄 Duplicates Handling
During migration, if multiple skills shared the exact same name across different repositories:
- **Exact Match**: Only one copy was kept to reduce redundancy.
- **Differences Found**: The contents were merged within the same file under `<!-- MERGED CONTENT FROM DUPLICATE SOURCE: path -->` to ensure no context was lost.
